== Description ==

Mocean Advertisement Plugin is a plugin for wordpress websites and provides a simple, straightforward way to place ads from an Mocean advertisement server on a website.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the zip file to the /wp-content/plugins/ directory and unzip. 
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to your Appearance -> Widgets and add 'Mocean Advertisement' widget to your page.
4. To see the click rates are available at Settings -> Mocean Stats
5. Error logs are visible at Settings -> Mocean Error Logs
